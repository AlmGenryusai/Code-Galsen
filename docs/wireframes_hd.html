Typographie. Fraunces pour les titres (variable, axes SOFT et WONK pour le caractère éditorial) ; Plus Jakarta Sans en texte courant ; JetBrains Mono pour les méta. À conserver en woff2 self-hosted, preload critique.

Motifs récurrents. Pointillés (référence aux marquages routiers), arcs et cercles ouverts (panneaux), nombres en italique pour les statistiques, cercles avec flèche (CTA principal). Texture grain SVG à très faible opacité sur tous les fonds pour échapper au plat numérique.

Implémentation Next.js.

  • Variables CSS dans app/globals.css, copiables tels quels.
  • Composants shadcn à étendre : Button, Input (OTP), Card, Progress, Sheet.
  • Bundle fonts : Fraunces ~140 Ko (subset latin), Plus Jakarta Sans ~80 Ko, JetBrains Mono ~60 Ko. Cible total < 300 Ko, retirer les axes inutilisés.
  • Tous les écrans en mode portrait 380×800 — pas de support tablette/desktop en MVP.
  • Animations : entrées staggered au chargement, micro-interactions au tap (Motion ou CSS pur). Pas de scroll-triggered en MVP, coût performance Android low-end.
